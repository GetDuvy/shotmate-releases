## 🌟 Overview

**ShotMate Desktop** is the official native desktop application designed specifically for professional event photographers, videographers, and creators. ShotMate eliminates the frustrating limitations of traditional browser-based uploads, allowing you to upload thousands of high-resolution RAW/JPEG photos and 4K videos smoothly while tracking your sales in real time.

---

## ⚡ Why ShotMate Desktop is Superior to Web Upload

Uploading large volumes of high-resolution images (50MB+ RAW files) or multi-gigabyte video files via web browsers often leads to tab crashes, slow canvas compression, and lost progress. **ShotMate Desktop** provides a dedicated native engine built for speed, stability, and scale.

| Feature / Capability | 🌐 Traditional Web Upload | 🚀 ShotMate Desktop Client |
| :--- | :--- | :--- |
| **RAM & Memory Limits** | Constrained by browser memory limits (crashes at 2GB–4GB RAM). | **Unlimited Native Batch Capacity**: Smoothly handles 100,000+ high-res files without crashing. |
| **Image & Video Compression** | Slow single-threaded browser canvas processing. | **Multi-Threaded Hardware Compression**: Blazingly fast local pre-compression before upload. |
| **Connection Stability** | Interrupted if browser tab is closed, refreshed, or connection drops. | **S3 Multipart Direct Upload**: Background chunking with auto-retry and seamless pause/resume. |
| **Folder & Directory Support** | Clunky file pickers with browser permission popups. | **Seamless Native Drag-and-Drop**: Drop nested folder trees directly into the app instantly. |
| **Offline Workflows** | Requires constant active internet connection to prepare uploads. | **Offline Batch Staging**: Stage files, edit EXIF metadata, and build upload queues anytime. |
| **Notifications & Sales Alerts** | In-browser alerts (missed when browser is minimized/closed). | **Native Desktop Toasts**: Real-time sales and upload alerts delivered directly to your OS taskbar. |

## ✨ Features & Capabilities

### 📤 High-Speed Multi-Batch Uploader
- **Unlimited Batch Queueing**: Create, organize, and upload multiple photo and video batches simultaneously.
- **Drag & Drop Folder Hierarchy**: Simply drag entire folder trees into the application to auto-structure your upload.
- **EXIF & IPTC Metadata Preservation**: Automatically retains camera settings (ISO, aperture, shutter speed, lens info, GPS location, timestamps).
- **Local Media Compression**: Built-in image and video compression reduces raw file sizes prior to transmission, saving hours of upload time.
- **Checksum Verification**: Automatic MD5/SHA256 file hashing guarantees zero corrupted uploads or duplicate files.

### 📊 Real-Time Creator Sales Dashboard
- **Live Sales Tracking**: Monitor photo sales, customer orders, and revenue potential directly from your desktop.
- **Instant Desktop Toasts**: Receive immediate system notifications whenever a photo or album sells.
- **Upload & Sales History**: Access comprehensive logs of past uploads, active galleries, and download statistics.

### 🔐 Licensing & Seamless Updates
- **Secure License Key Activation**: One-click activation with machine binding.
- **Automated Background Updates**: Automatic background checks ensure you are always running the latest version with the newest features.

### 🌐 Multilingual Support
- Switch between **English** and **Indonesian (Bahasa Indonesia)** with full localization across all screens.

## 🖥️ System Requirements

- **Operating System**: Windows 10 / Windows 11 (64-bit)
- **Processor**: Intel Core i3 / AMD Ryzen 3 or higher
- **Memory**: 4 GB RAM minimum (8 GB+ recommended for heavy 4K video compression)
- **Storage**: 500 MB free space for application installation

## 💻 Installation

1. Download the latest installer (`ShotMate_x64-setup.exe`) from the official release page.
2. Run the installer and follow the quick setup wizard.
3. Launch **ShotMate**, activate your creator license key, and begin uploading!
